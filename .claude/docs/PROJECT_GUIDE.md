# English Learning Website

## Goal

Build an English learning website for Vietnamese learners.

The website must be:

- Modern UI
- Fast
- Mobile responsive
- Easy to maintain
- Easy to add new lessons
- SEO friendly
- Beautiful animations
- Support dark mode
- Local first

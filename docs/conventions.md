# Quy ước phát triển

## Ngôn ngữ

- Giao tiếp & tài liệu: **tiếng Việt**.
- Nội dung học: song ngữ **Anh–Việt**.
- Tên biến, hàm, file: **tiếng Anh**.

## Code style

- Dùng `const`/`let`, không dùng `var`.
- Đặt tên rõ nghĩa; hàm làm một việc.
- Comment khi logic không hiển nhiên, viết bằng tiếng Việt.

## Git

- Commit message ngắn gọn, thì hiện tại: "thêm trang đăng nhập".
- Không commit/push khi chưa được yêu cầu.
- Mỗi tính năng một nhánh riêng khi có thể.

## Nội dung bài học

- Mỗi từ vựng phải có phiên âm và ít nhất một ví dụ song ngữ.
- Quiz: đáp án phải nằm trong danh sách options.
- Phân loại cấp độ theo CEFR: A1, A2, B1, B2, C1, C2.

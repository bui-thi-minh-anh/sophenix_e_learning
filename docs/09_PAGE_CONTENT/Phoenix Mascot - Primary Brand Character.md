# Phoenix Brand Mascot

This phoenix is the official mascot of the project.

Usage:
- Homepage hero section
- Loading animation
- Empty states
- Error pages (404/500)
- Achievement badges
- Navigation illustrations
- AI assistant avatar
- Marketing materials

Design rules:
- Keep the same chibi style.
- Use consistent proportions and colors.
- Create different expressions instead of redesigning the mascot.

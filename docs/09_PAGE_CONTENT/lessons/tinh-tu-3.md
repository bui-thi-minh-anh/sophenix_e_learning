# Tính từ (3)

## VII. Tính từ đơn, tính từ ghép

### 1. Tính từ đơn (Noun → Adj)

| Cách | Ví dụ |
|------|-------|
| Noun + y | rain → rainy, wind → windy, sun → sunny, snow → snowy, fog → foggy |
| Noun + ly | day → daily, man → manly, friend → friendly, love → lovely |
| Noun + ful | care → careful, beauty → beautiful, harm → harmful |
| Noun + less | care → careless, home → homeless, harm → harmless |
| Noun + en | gold → golden, wool → woollen, wood → wooden |
| Noun + some | trouble → troublesome, quarrel → quarrelsome |
| Noun + ish | book → bookish, girl → girlish, child → childish |
| Noun + ous | humour → humorous, danger → dangerous |

### 2. Tính từ ghép

| Cách | Ví dụ |
|------|-------|
| Adj + adj | a dark-blue coat, a red-hot iron bar |
| Noun + adjective | a snow-white face, the oil-rich country |
| Noun + participle | a horse-drawn cart, a heart-breaking story |
| Adjective + participle | ready-made shirt, a good-looking girl |
| Adverb + participle | a newly-built house, a well-dressed man |
| Noun + noun-ed | a tile-roofed house |
| Adjective + noun-ed | a dark-haired girl |
| Nhóm từ (group of words) | a twenty-year-old girl, a twelve-chapter novel, an eight-day trip, an air-to-air missile |

## VIII. Hậu tố, tiền tố của tính từ

### 1. Hậu tố (Adj endings)

-ent, -ant, -ful, -less, -ic, -ive, -ous, -able, -ible, -al, -ory, -ary, -y, -ly, -ate, -ish.

_(Ví dụ: independent, arrogant, beautiful, careless, classic, passive, dangerous, reliable, audible, central, compulsory, necessary, happy, friendly, accurate, foolish…)_

### 2. Tiền tố (Prefix — thường tạo nghĩa phủ định)

| Tiền tố | Ví dụ |
|---------|-------|
| non- | non-smoking, non-alcoholic |
| inter- | international, intercontinental |
| il- | illegal, illegible, illiterate, illogical |
| ir- | irreplaceable, irregular, irresponsible |
| un- | uncomfortable, unsuccessful, unhappy, unlucky, unemployed |
| in- | inconvenient, inedible, incomplete, independent |
| im- | immature, impatient, impossible, impractical |

## IX. Thứ tự của các tính từ chỉ phẩm chất (OSASCOMP)

**O**pinion → **S**ize → **A**ge → **S**hape → **C**olor → **O**rigin → **M**aterial → **P**urpose

| | Loại | Ví dụ |
|---|------|-------|
| O | Opinion (ý kiến) | beautiful, excellent |
| S | Size (kích cỡ) | big, small |
| A | Age (tuổi) | old, new |
| S | Shape (hình dáng) | round, oval, square |
| C | Color (màu) | blue, red, yellow |
| O | Origin (nguồn gốc) | Vietnamese, Japanese |
| M | Material (chất liệu) | cotton, woollen |
| P | Purpose (mục đích) | walking (stick), sleeping (bag), reading (lamp) |

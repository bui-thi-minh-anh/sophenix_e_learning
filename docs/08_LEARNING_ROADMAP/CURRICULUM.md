# English Learning Curriculum

## Goal

This website teaches English grammar from absolute beginner (A0) to upper-intermediate (B2).

Lessons must always follow this order.

Never skip lessons.

Every lesson should include:

- Explanation
- Formula
- Usage
- Examples
- Common mistakes
- Notes
- Quiz
- Exercises
- Summary
- Related lessons

# Page Overview

The Speaking page helps learners improve English pronunciation and speaking confidence through a progressive learning journey.

The page should feel encouraging, interactive, and premium while maintaining the same design language as the Home, Lessons, Vocabulary, Listening, and Reading pages.

Dark Mode

Mascot:
Cute Chibi Phoenix holding a microphone.

# Design Goal

The page should look like a premium English learning platform inspired by:

- Duolingo
- ELSA Speak
- Headspace
- Linear

Focus on:

- Clean hierarchy
- Large spacing
- Premium illustrations
- Clear learning progression
- Encouraging learning experience

Do not redesign the global layout.

Keep consistency with the existing Sophenix UI.

# Overview

Design a premium English Writing page for Sophenix.

This page focuses on helping learners practice English writing from basic writing to TOEIC and IELTS.

The page should look modern, premium, motivating, and consistent with the existing Sophenix design.

Dark theme.

Do NOT redesign the sidebar or top navigation.
